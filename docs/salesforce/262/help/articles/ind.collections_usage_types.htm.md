# Billing Considerations for Collections and Recovery

Use of certain Collections and Recovery feature impacts the consumption of credits used for billing in these usage types.

USAGE TYPE	DESCRIPTION	NOTES
Collections	Usage is calculated based on the number of collection plan records created for the Collections and Recovery application.	

Collections usage isn’t reported in Digital Wallet. Your usage statement will be sent via email on a monthly basis.

For details about how the Collections usage type is billed, see the Collections Rate Card.

For more information on how usage is billed, refer to your contract or contact your account executive.
