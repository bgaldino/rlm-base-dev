# Set Up Document Generation Templates for Collections and Recovery Communication

Enable Design Document Templates, clone and customize predefined document generation templates, and generate collections-related documents by using these templates.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
Enable Design Document Templates
To use Design Document Template to create DOCX, PPT, and Microsoft 365 Word templates in Salesforce, turn on Design Document Templates in Salesforce.
Clone and Customize the Predefined Document Generation Templates
Clone and customize the predefined Document Generation templates according to your business requirements.
Generate Collections-Related Documents with Document Templates
You can generate documents from the document templates either by using Omniscript, or the document generation batch process.
