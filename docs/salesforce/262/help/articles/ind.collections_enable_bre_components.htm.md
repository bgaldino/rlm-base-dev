# Enable Business Rules Engine Components for Collections and Recovery

Enable the Business Rules Engine components to allow users to access and configure context definitions, expression sets, and lookup tables, which helps you to determine the segments for collection plans.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To turn on Context Definition:	Context Service Admin permission set
To turn on Industries Cloud Common Decision Tables Access:	Rule Engine Designer permission set
To turn on Actionable Event Orchestration:	Actionable Event Orchestration Designer permission set
From Setup, in the Quick Find box, enter Context Service Settings, and then turn on Context Definitions.
From Setup, in the Quick Find box, enter Business Rules Engine Settings, and then turn on Industries Cloud Common Decision Tables Access.
From Setup, in the Quick Find box, enter Actionable Event Orchestration Settings, and then turn on Actionable Event Orchestration.
