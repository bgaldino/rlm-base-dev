# Enable Financial Account Management Standard Objects and Configure Context Definitions

To use the Financial Account and related standard objects, enable Financial Account Management Standard Objects. To access and configure context definitions, enable Context Definitions.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To enable Financial Account Management Standard Objects:	

Industry Service Excellence OR Industry Sales Excellence

AND

OmniStudio User

AND

Financial Services Cloud Extension OR FSC Foundation OR FSC Service OR FSC Sales


To enable Context Definition:	Context Service Admin permission set
In Setup, find and select Financial Account Settings, and then turn on Financial Account Management Standard Objects.
In Setup, find and select Context Service Settings, and then turn on Context Definitions.
Make sure that you set up a compliance application that can verify the call compliance rules based on your business requirements. You can set up the Salesforce compliance application, Process Compliance Navigator, or a third-party application.
