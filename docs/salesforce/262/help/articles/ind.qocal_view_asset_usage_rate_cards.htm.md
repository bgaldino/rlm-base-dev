# View Rate Cards for Usage-Based Assets

To understand the financial and resource parameters of a customer's purchase, view the usage resources associated with their usage-based assets. The rate card provides detailed information on included resources, quantities rated to the customer, rates per resource, and tier pricing.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: Enterprise, Unlimited, and Developer Editions of Revenue Cloud where Transaction Management is enabled
USER PERMISSIONS NEEDED
To view asset rate cards:	View on asset

The Usage Rates tab shows information only for usage-based assets. The tab remains blank when the asset isn't usage-based. Usage rate information becomes visible after an admin adds the Usage Rate component to the Asset record page. Rate information is read-only.

From the Asset related list on an account or from the Managed Assets Viewer on an account, select an asset.
Click Usage Rates.
