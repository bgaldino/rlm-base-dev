# Setup and Configuration of Collections and Recovery Portal

Use the Collections and Recovery Portal prebuilt template to create a dedicated portal for the collections and recovery specialists of third-party agencies.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.

The dedicated Collections and Recovery portal helps collections specialists monitor and manage operations, create promise-to-pay agreements, and keep track of customer visits—all without leaving the portal.

Clone and Assign Permissions Sets for Collections and Recovery Portal
To control access to the Collections and Recovery features, create a profile for portal users with the appropriate permissions by cloning a community user profile. Assign object permissions. Clone the Collections and Recovery for Experience Cloud permission set and assign it to your portal users. This permission set gives the portal users the license and permissions to access the Collections and Recovery Portal and the prebuilt quick actions and flows.
Create an Experience Site by Using the Collections and Recovery Portal Template
Give your customers access to the Collections and Recovery features through the Collections and Recovery Experience site.
Create and Configure Action Launcher Deployment for Collections and Recovery Portal
Create an Action Launcher deployment to include the Collection Plan object, and related actions and flows. Collections specialists can use the actions from the Action Launcher on a collection plan record details page.
