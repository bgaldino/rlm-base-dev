# Set Up Collection Campaigns

Create data streams, identity resolution rulesets, data graphs, and segments by using the Collections Data Kit so that collections managers can easily use collections data to create and manage collection campaigns, track the campaign's performance, and notify customers of their payment status.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create data streams, identity resolution rulesets, data graphs, and segments:	Data Cloud Admin
To create and manage collection campaigns:	

Data Cloud Admin

AND

Marketing Cloud Admin

Assign the required permissions, access the setup assistant, and enable Marketing Cloud.
Create a data stream by using Collections Bundle.
From the App Launcher, find and select Data Cloud.
On the Data Streams tab, click New.
Select Salesforce CRM as the connected source, and click Next.
In the Standard Data Bundles section, select Collections Bundle.
Review the data fields, and click Next.
Click Deploy.
Create an identity resolution ruleset by using the Collections Data Kit.
From the App Launcher, find and select Data Cloud.
On the Identity Resolutions tab, click New.
Select Install from Datakits, and click Next.
Select Collection Identity Resolution Ruleset, and click Next.
Enter a ruleset name, and save the changes.
Create a data graph by using the Collections Data Kit.
From the App Launcher, find and select Data Cloud.
On the Data Graphs tab, click New.
Select Use a Data Kit, and click Next.
Select Collections Email Data Graph, and click Next.
Review the details, and click Save and Build.
Create a segment by using the prebuilt segment template that's shipped with the Collections Data Kit.
From the App Launcher, find and select Data Cloud.
On the Segments tab, click New.
Select DataKit Segment, and click Next.
Select Early Stage Collections Sample, and click Next.
Select the publish type and publish schedule, and save the changes.
After you create the data stream, identity resolution ruleset, data graph, and a segment, collections managers can readily use collections data to create Collections campaigns.
Create a collections campaign, and get started with the collections activities.
