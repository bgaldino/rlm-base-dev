# Create an Experience Site by Using the Collections and Recovery Portal Template

Give your customers access to the Collections and Recovery features through the Collections and Recovery Experience site.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create an Experience site by using Collections and Recovery Portal template:	

Create and Set Up Experiences

AND

View Setup and Configuration


To customize the Experience site:	

Be a member of the portal AND Create and Set Up Experiences

OR

Be a member of the portal AND View Setup and Configuration AND an Experience Cloud admin, publisher, or builder in that portal

Before you create an Experience Cloud site, enable Digital Experiences in Salesforce.

Use the prebuilt portal template, Collections and Recovery Portal to create an Experience Cloud site so that collections specialists of third-party agencies can use this portal to monitor and manage collection and recovery operations.

In Setup, find and select All Sites.
Click New, and select the Collections and Recovery Portal template.
Click Get Started.
Enter a name for the site.
Enter the name of your site as the URL.
This name is appended to the domain that you created when you enabled Digital Experiences. For example, if your site’s domain name is cumuluscapital.com and if you’re creating a collections and recovery site, you can enter CollectionsAndRecovery to create the URL, cumuluscapital.com/CollectionsAndRecovery.
Click Create.
Click Administration, and then under Settings, click Activate.
Under Members, from the Search dropdown, select Customer.
From Available Profiles, add your cloned community profile to the Selected Profiles list.
Save your changes.
