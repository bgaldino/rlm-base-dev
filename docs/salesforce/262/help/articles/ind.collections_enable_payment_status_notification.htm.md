# Enable Payment Status Notification

Automatically send emails to borrowers to notify them of their payment status when a payment transaction event is created.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To enable the Payment Status Notification setting:	

Collections and Recovery Admin

AND

Data Cloud Admin

AND

Marketing Cloud Admin

From Setup, in the Quick Find box, enter Collections, and then select Collections Settings.
Enable Payment Status Notification.
