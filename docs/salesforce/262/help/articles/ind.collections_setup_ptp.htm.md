# Set Up Promise to Pay

Customize the prebuilt flow that helps collections specialists create promise to pay agreements for borrowers. A Promise to Pay is a commitment by the delinquent borrower or customer to pay a specified amount by a certain date.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
Customize the Prebuilt Flow to Create Promise to Pay Agreements
Customize the Create Promise to Pay prebuilt flow. This flow is triggered through the Create Promise to Pay action on a collection plan page.
Configure Maximum Promise to Pay Count
Set a limit on the number of promises to pay for a collection plan.
