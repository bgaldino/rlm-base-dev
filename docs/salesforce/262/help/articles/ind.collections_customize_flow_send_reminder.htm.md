# Customize the Prebuilt Flow Template to Send a Reminder to Borrower with Payment Link

Create a flow by using the Send Reminder with Payment Link flow template to send an email to the borrower with the payment link.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create and customize a flow by using the prebuilt flow template:	

Collections and Recovery Admin permission set

AND

Customize Application permission

AND

Manage Flow permission

AND

Run Flows permission

From Setup, in the Quick Find box, enter Flows, and then select Flows.
Click Collections: Send Reminder with Payment Link.
Click Save as New Version.
NOTE If you plan to clone the flow, make sure that you don't change the API name.
Customize the flow according to your business needs.
Save your changes and activate the flow.
