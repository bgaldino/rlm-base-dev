# Create Cases for Collection Plans

As a collections specialist, create and update cases directly on the collection plan record details page, streamlining workflows and enhancing efficiency in managing and resolving collections.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create a case manually for a collection:	Collections and Recovery Specialist

Before you create a case for a collection plan:

Make sure that field-level security for the case source field on the Case object is set. See Set Field-Level Security for Case Source.
Make sure that the case source field is added to the case information section of the case page layout. See Customize the Case Lightning Record Page.
From the App Launcher, find and select Collections.
To view a collection plan record details page, from the collection plans section, click a collection plan.
To launch the action, on the collection plan record details page, click Create Case.
Enter the case details and save your changes.
