# Configure Automatic Payment Status Notifications

Customize the prebuilt flow to automatically send payment status emails to customers. To use this flow, make sure that you have set up Data 360 and Marketing Cloud in your org.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
Enable Payment Status Notification
Automatically send emails to borrowers to notify them of their payment status when a payment transaction event is created.
Customize the Prebuilt Flow to Send Payment Status Notifications
Customize the prebuilt draft flow for payment status notifications to include payment success or failure messages.
