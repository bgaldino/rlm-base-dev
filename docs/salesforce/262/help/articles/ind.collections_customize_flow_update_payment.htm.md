# Customize the Prebuilt Flow to Update the Payment Details

This flow is triggered when a payment intent event is created for a successful transaction, and updates the payment received details in the corresponding collection plan or payment schedule item.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To customize the prebuilt flow:	

Collections and Recovery Admin permission set

AND

Customize Application permission

From Setup, in the Quick Find box, enter Flows, and then select Flows.
Click Collections: Update Payment Details.
Click Save As New Version.
NOTE If you plan to clone the flow, make sure that you don't change the API name.
Customize the flow according to your business requirements.
Save your changes and activate the flow.
