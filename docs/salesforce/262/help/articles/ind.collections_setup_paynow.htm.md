# Set Up Pay Now for Collections and Recovery

To enable Payments, first turn on Digital Experiences and set up an Experience Cloud site, and then set up Pay Now. Salesforce PayNow helps your collections specialists to create and share secure payment links that direct your customers to a payment page. Customers can enter their contact and payment details to complete the transaction.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To set up Pay Now by using Commerce Setup Assistant:	

Collections and Recovery Admin

AND

FSC Sales OR FSC Service

AND

Industry Service Excellence

AND

Payments Admnistrator


To change digital experience settings:	Customize Application
Before you create an Experience Cloud site for your customers, Enable Digital Experiences.
Use the Commerce Setup Assistant to set up Pay Now.
Make sure that you have the required licenses before proceeding.
