# Customize the Case Lightning Record Page

To make sure that the collections specialists can view and edit case source details and to verify that the case is correctly linked to the collection plan, add the case source field to the case information section of the case page layout.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To customize the Case Lightning record page:	

Collections and Recovery Admin permission set

AND

Modify All Data permission

AND

Customize Application permission

From the object management settings for the Case object, go to Case Page Layouts.
Open the page layout that you want to edit.
On the palette, select Fields.
Drag the case source field from the object palette to the Case Information section.
Save your changes.
