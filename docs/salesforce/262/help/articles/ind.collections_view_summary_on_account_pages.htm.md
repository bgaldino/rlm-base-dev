# View Metrics and Linked Collection Plans on Account Pages

As collections specialists, you can quickly identify high-priority collection accounts and strategically plan collection activities by reviewing key metrics on the account record pages. These metrics include the total initial due amount, the current total due, the total payments received, and the average days past due. You can also view all collection plans associated with the account on the account record details page.

REQUIRED EDITIONS

Make sure that your org admin has set up the required prerequisites so that you can view the Collections metrics and linked collection plans on account record pages.

From the App Launcher, find and select Collections.
To view the key Collections metrics and all the collection plans linked to an account, go to Accounts, and open an account record details page.
