# Create and Configure Action Launcher Deployment for Collections and Recovery Portal

Create an Action Launcher deployment to include the Collection Plan object, and related actions and flows. Collections specialists can use the actions from the Action Launcher on a collection plan record details page.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create an Action Launcher deployment:	Industry Service Excellence permission set
From Setup, in the Quick Find box, enter Action Launcher, and then select Deployments.
Click New Deployment.
Name your deployment.
From Available Objects, find and select the Collection Plan object and add it to Selected Objects.
Select the Create Promise to Pay, Capture Collections-Related Interaction, Email, New Note, and Create a Case actions.
Save your changes.
To help collections specialists launch the actions related to Collections, add the Action Launcher component to the collection plan Lightning record page, and select the Action Launcher deployment that you created earlier.
