# Create and Configure Prioritized Lists of Collection Plans with Actionable List Definitions

As a collections manager, use actionable list definitions to create actionable lists. Specify list filtering criteria, such as due amount, days past due, or collection plan segment. After you create an actionable list, assign it to collections specialists so that they can get started with the collections activities quickly.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create and configure actionable lists of collection plans:	

Collections and Recovery Specialist permission set

AND

Actionable Segmentation permission set

AND

Query for Datapipelines User permission set

AND

View Setup and Configuration permission

Before creating a list, make sure that your Salesforce admin has already created, activated, run the required actionable list definition. The actionable list definition creates a CRM dataset that contains collection plan and related records. You can use this dataset associated with the list definition to create prioritized list of collection plans. See Set Up Actionable Segmentation for Collections and Recovery.

From the App Launcher, find and selectActionable Lists, and then click New.
