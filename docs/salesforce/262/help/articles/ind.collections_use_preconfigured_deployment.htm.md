# Configure Action Launcher Deployment for Collections and Recovery

Use the preconfigured Action Launcher deployment for Collections to help your collections specialists launch actions related to Collections. Make the Action Launcher component available to your users by adding the component to the collection plan Lightning record page.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To customize preconfigured action launcher deployment:	

Industry Service Excellence

AND

Collections and Recovery Admin

Collections includes a preconfigured action launcher deployment, Collection Plan Processes. You can customize it according to your business requirements.

To launch actions, such as Create Promise to Pay and Generate and Send Payment Link on collection plan details page, use the preconfigured action launcher deployment, Collection Plan Processes. See Use a Standard Action Launcher Deployment.
To help collections specialists launch the actions related to Collections, add the Action Launcher component to the collection plan Lightning record page, and select the Collection Plan Processes Action Launcher deployment. See Add the Action Launcher Component to a Lightning Record Page.
