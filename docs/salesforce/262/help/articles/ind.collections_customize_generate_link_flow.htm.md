# Customize the Prebuilt Flow Template to Generate and Send a Payment Link

Help collections specialists create and send payment links to borrowers. The Generate and Send Payment Link prebuilt flow template helps a collections specialist to gather payment details, such as payment amount, payment processing platform, whether it’s a one-time or multiple-use payment link and its validity. The flow also generates a payment link and emails it to the borrower if an email address is provided, or shows the payment link details on the screen. Customize this flow according to your business needs.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To create and customize a flow by using the prebuilt flow template:	

Collections and Recovery Admin permission set

AND

Customize Application permission

AND

Manage Flow permission

AND

Run Flows permission

From Setup, in the Quick Find box, enter Flows, and then select Flows.
Click Collections: Generate and Send Payment Link.
Click Save as New Version.
NOTE If you plan to clone the flow, make sure that you don't change the API name.
Customize the flow according to your business needs.
Save your changes and activate the flow.
