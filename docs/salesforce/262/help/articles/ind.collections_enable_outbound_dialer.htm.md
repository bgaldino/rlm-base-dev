# Turn on Features to Set Up Outbound Dialer for Collections and Recovery

To set up outbound dialer integration for Collections, turn on Collections Outbound Dialer, Data Pipelines, and Actionable Segmentation.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To turn on Data Pipelines:	Data Pipelines Base User
To turn on Actionable Segmentation:	Actionable Segmentation
To turn on Collections Outbound Dialer:	Collections and Recovery Admin
Turn on Data Pipelines and Actionable Segmentation.
From Setup, in the Quick Find box, find, and select Collections Settings.
Turn on Collections Outbound Dialer.
