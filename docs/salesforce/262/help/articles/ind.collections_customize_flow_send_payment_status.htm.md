# Customize the Prebuilt Flow to Send Payment Status Notifications

Customize the prebuilt draft flow for payment status notifications to include payment success or failure messages.

REQUIRED EDITIONS
Available in: Lightning Experience
Available in: View product and edition availability.
USER PERMISSIONS NEEDED
To customize the prebuilt flow:	

Collections and Recovery Admin permission set

AND

Customize Application permission

AND

Data Cloud Admin permission set

AND

Marketing Cloud Admin permission set

To use this flow:

Set up Data 360 and Marketing Cloud in your org.
Enable the Payment Status Notification setting in Collections Setup.
From Setup, in the Quick Find box, enter Flows, and then select Flows.
Click Collections: Send Payment Status Email.
Click Save as New Version.
To send payment success and payment failure messages to your borrowers, configure the two Send Email Message action elements in the flow.
Save your changes and activate the flow.
