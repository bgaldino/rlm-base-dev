# Data Processing Engine Invocable Actions

Run an active Data Processing Engine definition. For more
information on custom invocable actions, see **[REST API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.262.0.api_rest.meta/api_rest/resources_actions_invocable.htm)** and **[Actions Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.262.0.api_action.meta/api_action/actions_intro.htm)**.

- **[Data Processing Engine Actions](./dpe_actions_run_data_processing_engine.htm.md)**  
  Run an active Data Processing Engine definition. This action executes a Data Processing Engine definition asynchronously.
