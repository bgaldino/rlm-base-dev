# Decision Table APIs

Decision table APIs are RESTful APIs that are sometimes available as Apex classes
and methods.

- **[REST Reference](./decision_table_apis_rest_references.htm.md)**  
  You can access Decision Table APIs using REST endpoints. These REST APIs follow similar conventions as Connect REST APIs.
