# Query Context Definition Interfaces (GET)

Get the lists of metadata associated with context definition interfaces.

Resource
:   ```
    /connect/context-definition-interfaces
    ```

Resource example
:   ```
    https://yourInstance.salesforce.com/services/data/v66.0/connect/context-definition-interfaces
    ```

Available version
:   62.0

HTTP methods
:   GET

Response body for GET
:   [Context Definition
    Interface Metadata List](./connect_responses_context_definition_interface_metadata_list.htm.md "Output representation of the metadata list associated with the context definition interfaces.")
