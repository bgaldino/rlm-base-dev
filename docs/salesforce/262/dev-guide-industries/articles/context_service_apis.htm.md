# Context Service Business APIs

Context Service Business APIs are RESTful APIs that are sometimes
available as Apex classes and methods.

- **[REST Reference](./context_service_api_rest_references.htm.md)**  
  You can access the Context Service APIs using REST endpoints. These REST APIs follow similar conventions as Connect REST APIs.
