# Decision Matrix Lookup Basic Result

Results from a Decision Matrix lookup using a single
input.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| `error` | String | The message stating the reason for the error. | Small, 55.0 | 55.0 |
| `results` | [Variable Output](./connect_responses_variable_output.htm.md "Output representation of the input variables in an expression set.")[] | The list of outputs for the specified input variable values. | Small, 55.0 | 55.0 |
