# Rule Results

Output representation of the result of executing a single input in an
expression set.

| Property Name | Type | Description | Filter Group and Version | Available Version |
| --- | --- | --- | --- | --- |
| `error` | [Error Result](./connect_responses_error_result.htm.md "Output representation of the errors in the execution of an expression set.") | The details of the error resulting from executing a single input in an expression set. | Small, 55.0 | 55.0 |
| `results` | [Variable Map Output](./connect_responses_variable_map_output.htm.md "Output representation of the input variables in an expression set version.") | The result of the aggregation step in an expression set. | Small, 55.0 | 55.0 |
