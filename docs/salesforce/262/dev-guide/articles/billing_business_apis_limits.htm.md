# Billing Business API Limits

Learn about the default limits on the usage of the Billing business APIs.

|  |
| --- |
| Available in: Lightning Experience |
| Available in: **Enterprise**, **Unlimited**, and **Developer** Editions |

Refer to these default API limits and applicable recommendations to scale beyond these
default limits.

| Limit | Value | Scale Recommendations |
| --- | --- | --- |
| The number of billing schedules that can be processed by the [Create Invoices By Using Billing Schedules API](./connect_resources_create_invoices_from_billing_schedules.htm.md "HTML (New Window)"). | 200 | Use Invoice Scheduler to scale up to 2000 invoice lines. See [Invoice Scheduler](https://help.salesforce.com/s/articleView?id=ind.billing_guided_setup_invoice.htm&language=en_US "HTML (New Window)") or [Batch Invoice Scheduler (POST) API](./connect_resources_create_an_invoice_scheduler.htm.md "HTML (New Window)"). |
| The number of invoice lines that can be generated by the [Create Invoices By Using Billing Schedules API](./connect_resources_create_invoices_from_billing_schedules.htm.md "HTML (New Window)"). | 200 | Use Invoice Scheduler to scale up to 2000 invoice lines. See [Invoice Scheduler](https://help.salesforce.com/s/articleView?id=ind.billing_guided_setup_invoice.htm&language=en_US "HTML (New Window)") or [Batch Invoice Scheduler (POST) API](./connect_resources_create_an_invoice_scheduler.htm.md "HTML (New Window)"). |
| The number of billing schedules that can be processed by the [Recover Billing Schedule List API](./connect_resources_recover_billing_schedules.htm.md "HTML (New Window)"). | 200 | Use this API to follow the default API limits. |
| The number of invoices that can be processed by the [Apply Credit Memos API](./connect_resources_credit_memo_apply.htm.md "HTML (New Window)"). | 300 | Use this API recursively to apply left-over credits after 300 invoice lines. |
| The number of invoice lines that can be processed by the [Apply Credit Memo Lines API](./connect_resources_credit_memo_line_level_apply.htm.md "HTML (New Window)"). | 300 | Use this API iteratively for every 300 invoice lines to support more than 300 invoice lines. |
| The number of invoices that can be processed by the [Create and Apply Credit Memos API](./connect_resources_create_and_apply_a_credit_memo.htm.md "HTML (New Window)"). | 300 | Use this API to follow the default API limits. The limits include the count for both charge and tax lines. |
| The number of credit memo lines of type Charge that can be created by the [Create a Standalone Credit Memo API](./connect_resources_create_a_standalone_credit_memo.htm.md "HTML (New Window)"). | 300 | Use this API to follow the default API limits. The limits include the count for both charge and tax lines. |
| The number of invoice lines that can be processed by the [Convert Negative Invoice Lines to Credits API](./connect_resources_convert_negative_invoice_lines_to_credit.htm.md "HTML (New Window)"), which excludes the number of associated invoice line taxes. | 300 | Use this API to follow the default API limits. The limits include the count for invoice charge lines only. |
| The number of billing transaction items that can be processed by the [Create Billing Schedules for Orders API](./connect_resources_create_billing_schedules.htm.md "HTML (New Window)"). | 1000 | This API supports creation of 1000 order lines as billing schedules. |
| The number of reference IDs that can be processed by the [Suspend Billing API](./connect_resources_suspend_billing.htm.md "HTML (New Window)") and the [Resume Billing API](./connect_resources_resume_billing.htm.md "HTML (New Window)"). | 200 | Use this API to follow the default API limits. |
| The number of invoice lines that's supported by the [Invoice Draft to Posted Status API](./connect_resources_draft_to_posted_invoice.htm.md "HTML (New Window)"). | 200 | Use this API to follow the default API limits. |
| The number of records that can be processed by the [Invoice Ingestion API](./connect_resources_invoices_ingestion.htm.md "HTML (New Window)"). | 500 | Use this API to follow the default API limits. The limits include the count for all invoices, invoice lines, taxes, and address group records. |
| The number of invoice lines or billing schedules that can be processed by the [Invoice Preview API](./connect_resources_preview_invoices.htm.md "HTML (New Window)"). | 200 | Use this API to follow the default API limits. |
| The number of invoice lines that can be processed by the [Void a Posted Invoice API](./connect_resources_void_a_posted_invoice.htm.md "HTML (New Window)"). | 2000 | Use this API to follow the default API limits. |
| The number of invoices that can be processed by the [Posted Invoice List Write-Off API](./connect_resources_write_off_invoices.htm.md "HTML (New Window)"). | 300 | Use this API to follow the default API limits. |
| The number of invoice lines that can be generated on an invoice by the [Batch Invoice Scheduler (POST) API](./connect_resources_create_an_invoice_scheduler.htm.md "HTML (New Window)").  The number of records that can be recovered from a failed invoice run by the [Batch Invoice Run Recovery (POST) API](./connect_resources_recover_errored_invoices_batch_run.htm.md "HTML (New Window)"). | 2000 | Use this API to support the creation or recovery of up to 2,000 invoice lines on a single invoice. |
| The number of invoice lines that's supported by the [Batch Invoices Draft to Posted Status (POST) API](./connect_resources_batch_draft_invoices_to_posted.htm.md "HTML (New Window)"). | 2000 | Use this API to support 2000 invoice lines with no limit on the number of supported invoices. |
| The number of invoices that's supported by the [Tax Calculation API](./connect_resources_void_a_posted_invoice.htm.md "HTML (New Window)"). | 1 | Use this API to follow the default API limits. |
| The number of invoice lines that's supported by the [Tax Calculation API](./connect_resources_void_a_posted_invoice.htm.md "HTML (New Window)"). | 500 | The request can include 500 invoice lines. To avoid limit-related issues, test your TaxEngineAdapter Apex interface’s implementation to make sure that it adheres to the [Apex limit for total heap size](https://developer.salesforce.com/docs/atlas.en-us.262.0.salesforce_app_limits_cheatsheet.meta/salesforce_app_limits_cheatsheet/salesforce_app_limits_platform_apexgov.htm "HTML (New Window)"). |
| The number of invoice lines that's supported by the Tax Calculation API interface when used with the [Invoice Creation API](./connect_resources_create_invoices_from_billing_schedules.htm.md "HTML (New Window)"). | 200 | The request can include 200 invoice lines. To avoid limit-related issues, test your TaxEngineAdapter Apex interface’s implementation to make sure that it adheres to the [Apex limit for total heap size](https://developer.salesforce.com/docs/atlas.en-us.262.0.salesforce_app_limits_cheatsheet.meta/salesforce_app_limits_cheatsheet/salesforce_app_limits_platform_apexgov.htm "HTML (New Window)"). |
| The number of invoice lines that's supported by the Tax Calculation API interface when used with the [Invoice Batch Run API](./connect_resources_batch_draft_invoices_to_posted.htm.md "HTML (New Window)"). | 2000 | The request can include 2000 invoice lines. To avoid limit-related issues, test your TaxEngineAdapter Apex interface’s implementation to make sure that it adheres to the [Apex limit for total heap size](https://developer.salesforce.com/docs/atlas.en-us.262.0.salesforce_app_limits_cheatsheet.meta/salesforce_app_limits_cheatsheet/salesforce_app_limits_platform_apexgov.htm "HTML (New Window)"). |
| The number of records that's supported by the [Payment Line Apply API](./connect_resources_void_a_posted_invoice.htm.md "HTML (New Window)"). | 1 | Payment supports application for either an Invoice or InvoiceLine record based on the settlement-level preferences. |
| The number of records that's supported by the [Payment Line Unapply API](./connect_resources_payment_line_unapply.htm.md "HTML (New Window)"). | 1 | Payment supports unapplication for either a PaymentLineInvoice or PaymentLineInvoiceLine record based on the settlement-level preferences. |
